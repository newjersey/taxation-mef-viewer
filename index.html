Translate Raw Data Back Into a Form

The IRS' Modern E-File (MEF) program uses XML files as the standard file format for tax return data. This tool allows users to transform XML into a human-readable, printable format, using the IRS' own stylesheets.

Stylesheets and code are availabe open-source on GitHub.
